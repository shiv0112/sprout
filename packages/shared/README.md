# sprout-shared

[![PyPI](https://img.shields.io/pypi/v/sprout-shared.svg)](https://pypi.org/project/sprout-shared/)
[![License](https://img.shields.io/badge/license-Apache_2.0-blue.svg)](https://github.com/aryabyte21/sprout/blob/main/LICENSE)

Shared specs, models, and helpers used by all [Sprout](https://github.com/aryabyte21/sprout)
packages — the tool spec model (`SproutToolSpec`), the runtime tool wrapper
(`SproutTool`), the `@sprout_tool` decorator, and the JSON Schema for `spec.yaml`.

You don't usually install this directly — install
[`sprout-registry-api`](https://pypi.org/project/sprout-registry-api/) instead and
this comes along as a dependency.
